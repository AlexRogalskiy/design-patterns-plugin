# Design Patterns plugin for IntelliJ IDEA
This plugin is meant to provide on the fly implementation of various design patterns.

## Currently supported design patterns
* Behavioral
  * Strategy
* Creational
  * Builder (Inner)
  * Factory
  * Singleton
* Structural
